How it works

Four layers. One binary. The Rust core does the work. Android keeps it alive.

01

Rust Runtime

Simon|AI binary compiled for aarch64-linux-android. Agent loop, HTTP gateway, SQLite memory, tool execution, and channel adapters.

features = ["agent-runtime", "gateway", "memory-sqlite", "tool-shell"]
02

Foreground Service

Kotlin service spawns the binary as a subprocess. START_STICKY keeps it alive. A subprocess crash restarts the service without killing the app.

ProcessBuilder(binary, "daemon", "-p", "18789").start()
03

Dashboard UI

Jetpack Compose app connects to the local REST API at localhost:18789. Full native Android experience.

http://127.0.0.1:18789 → ApiClient → Compose UI
04
